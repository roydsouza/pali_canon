# Mātika (Buddhist Lists Index)

**Navigation**: [[INDEX|Pali Canon Vault]] / [[matika/INDEX|Mātika]]

This directory contains systematic registers of Buddhist lists (*mātika*) compiled from the Pali Canon. Each list is stored in a dedicated file with Romanized Pali and item-by-item English translations, fully cross-referenced.

## Available Lists

*   **[[four_noble_truths|Four Noble Truths (Cattāri Ariyasaccāni)]]**
    *   The core diagnosis of the human condition and the path to liberation.
*   **[[noble_eightfold_path|Noble Eightfold Path (Ariyo Aṭṭhaṅgiko Maggo)]]**
    *   The path of practice leading to the cessation of suffering.
*   **[[three_marks|Three Marks of Existence (Tilakkhaṇa)]]**
    *   The universal characteristics of all conditioned phenomena (impermanence, suffering, non-self).
*   **[[five_aggregates|Five Aggregates (Pañcupādānakkhandhā)]]**
    *   The components that make up a sentient being's experience.
*   **[[dependent_origination|Dependent Origination (Paṭiccasamuppāda)]]**
    *   The twelve links describing the causal chain of rebirth and suffering.
*   **[[five_precepts|Five Precepts (Pañcasīla)]]**
    *   The basic code of ethics for Buddhist lay practitioners.
*   **[[five_hindrances|Five Hindrances (Pañca Nīvaraṇāni)]]**
    *   Obstacles in the mind that block concentration and wisdom.
*   **[[seven_awakening_factors|Seven Awakening Factors (Satta Bojjhaṅgā)]]**
    *   Qualities developed during meditation leading to enlightenment.
*   **[[four_foundations_of_mindfulness|Four Foundations of Mindfulness (Cattāro Satipaṭṭhānā)]]**
    *   The framework for developing mindfulness (body, feelings, mind, phenomena).
*   **[[eight_precepts|Eight Precepts (Aṭṭhaṅgasīla)]]**
    *   The code of ethics for lay practitioners during retreats and observance days.
*   **[[three_refuges|Three Refuges (Tisarana / Tiratana)]]**
    *   The foundational declaration of faith in the Triple Gem (Buddha, Dhamma, Sangha).
*   **[[ten_perfections|Ten Perfections (Dasa Pāramī)]]**
    *   The ten qualities cultivated on the path to liberation.
*   **[[four_sublime_states|Four Sublime States (Cattāro Brahmavihārā)]]**
    *   The qualities of loving-kindness, compassion, appreciative joy, and equanimity.
*   **[[five_spiritual_faculties|Five Spiritual Faculties (Pañcindriya)]]**
    *   The core spiritual faculties (faith, energy, mindfulness, concentration, wisdom).
*   **[[three_unwholesome_roots|Three Unwholesome Roots (Akusalamūla)]]**
    *   The three root poisons (greed, hatred, delusion) causing unwholesome states and rebirth.
*   **[[four_right_exertions|Four Right Exertions (Cattāro Sammappadhānā)]]**
    *   The effort to prevent/abandon unwholesome states and arise/maintain wholesome states.
*   **[[ten_fetters|Ten Fetters (Dasa Saṃyojanā)]]**
    *   The ten mental bonds that chain beings to the cycle of rebirth.
*   **[[seven_purifications|Seven Purifications (Satta Visuddhi)]]**
    *   The seven-stage path of purification leading to final extinguishment.
*   **[[five_powers|Five Powers (Pañcabala)]]**
    *   The five spiritual powers (faith, energy, mindfulness, concentration, wisdom) parallel to the faculties.
