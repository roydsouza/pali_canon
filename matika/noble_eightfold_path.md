# Noble Eightfold Path (Ariyo Aṭṭhaṅgiko Maggo)

**Navigation**: [[INDEX|Pali Canon Vault]] / [[matika/INDEX|Mātika]]

## The List

1. **Sammādiṭṭhi**  
   *Right view*
2. **Sammāsaṅkappo**  
   *Right intention / Right thought*
3. **Sammāvācā**  
   *Right speech*
4. **Sammākammanto**  
   *Right action*
5. **Sammāājīvo**  
   *Right livelihood*
6. **Sammāvāyāmo**  
   *Right effort*
7. **Sammāsati**  
   *Right mindfulness*
8. **Sammāsamādhi**  
   *Right concentration*
