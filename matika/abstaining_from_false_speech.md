# Abstaining from False Speech (Musāvādā veramaṇī)

**Navigation**: [[../INDEX|Pali Canon Vault]] / [[INDEX|Mātika]] / [[five_precepts|Five Precepts]] / [[eight_precepts|Eight Precepts]]

---

## Canonical Definition

> "Musāvādā veramaṇīsikkhāpadaṃ samādiyāmi..."
> 
> *“I undertake the training rule to abstain from false speech...”*

---

## Detailed Description

The fourth precept. It requires refraining from deliberate lying, deceit, and misrepresentation. It protects truthfulness, trust, and integrity in communication.
