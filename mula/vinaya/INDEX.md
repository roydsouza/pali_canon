# Vinaya Piṭaka — Mūla

**Navigation**: [[INDEX|Pali Canon Vault]] / [[mula/INDEX|Mūla]]

The Vinaya Piṭaka ("Discipline Basket") contains the rules and regulations for the monastic Saṅgha, and is the primary canonical source for the Buddhist precepts.

## Structure

| Division | Contents |
|---|---|
| **Suttavibhaṅga** | The Pātimokkha rules with origin stories and analysis |
| — Mahāvibhaṅga | The 227 rules for monks (Bhikkhus) |
| — Bhikkhunīvibhaṅga | The 311 rules for nuns (Bhikkhunīs) |
| **Khandhaka** | Regulations for communal life and procedures |
| — Mahāvagga | 10 chapters on admission, uposatha, rains retreat, etc. |
| — Cūḷavagga | 12 chapters on procedures, schisms, nuns' rules |
| **Parivāra** | Summary and classification of the rules |
